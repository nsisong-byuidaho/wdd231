Okoli Travels: Site Plan

Site Name

Okoli Travels

This name represents a travel agency brand that highlights personalized, adventurous, and culturally rich travel experiences. The name "Okoli" adds an authentic, personal feel that resonates with a global audience.

Site Purpose

The website aims to provide an engaging and informative platform for travelers looking to explore destinations across the globe. It will include destination highlights, travel package options, an interactive booking form, and customer testimonials. Visitors can explore curated tours, discover seasonal promotions, and access travel tips all in one spot.

Scenarios

  • What travel packages are available for summer vacation?
  • How do I book a trip or contact a travel advisor at Okoli Travels?

Color Schema

The website will use the following colors:

  • #003366 (Dark Blue) - Used for headings, navigation bar, primary elements and footer.
  • #00A8E8 (Bright Blue) - Used for accents, buttons, and highlights.
  • #F4F4F4 (Light Gray) - Used for background elements to ensure readability.

Typography

The website will use the following fonts:

  • Roboto - Used for headings and navigation items.
  • Roboto - Used for body text and paragraphs.
